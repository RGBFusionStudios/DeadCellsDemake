# DeadCellsDemake
This is a fan-made Dead Cells demake game built in the Godot engine. Music, code, and graphics made by "not_real", the lead developer of Shatterfall.

We do not have an association with Motion Twin, Evil Empire, or anyone who helped make Dead Cells. We only rightfully claim the art, music, and code of this game, but not the name of this game.
